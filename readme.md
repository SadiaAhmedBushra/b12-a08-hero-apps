## ProVault: Promoting Productive Apps

## Description
ProVault is a web application offering apps that help boost productivity. Via this web application, users may install programs straight from the platform and browse a well-categorised selection of apps. They can also perceive comprehensive data, like downloads, ratings, and reviews, through this web app. Dynamic charts for app ratings are available as well. It has a responsive design and local storage integration for managing installed apps.

![ProVault Screenshot](https://i.ibb.co/5Wm6WML1/provaultss.png)

## Live Link: https://b12-a08-provault-ph.netlify.app/

## Project Overview

**Objective:**  
Create a web application that offers a curated collection of productivity apps which users can browse, install, and manage directly from the platform. The app provides detailed information about each program including downloads, ratings, and reviews, alongside dynamic visualizations of rating distributions.

**Target Audience:**  
Productivity enthusiasts, remote workers, and general users looking for tools to enhance efficiency.

**Key Metrics:**  
- Number of app installs/uninstalls  
- User engagement with app details and ratings  
- Interaction with charts and search functionality  

**Deployment:**  
The application is deployed as a Single Page Application (SPA) on Netlify.

---

## Features

- Browse and search productivity apps.  
- View comprehensive app details, such as the description, downloads, ratings, reviews, and company name.  
- Install/uninstall apps and use local storage to manage installed apps.  
- Interactive charts that show the distribution of ratings for every app.  
- Toast alerts for actions in real time.  

---

## Technologies Used

- **React.js** – For building user interfaces.  
- **React Router** – For page navigation and dynamic routing.  
- **Tailwind CSS and Daisy UI** – For responsive styling.  
- **Recharts** – To display interactive charts.  
- **React Toastify** – For toast notifications.  
- **JavaScript (ES6+)** – As core programming language.  
- **HTML5 & CSS3** – For markup and base styling.

## Installation

Install the project dependencies for both server and client:

```bash
# Clone the repository
git clone https://github.com/SadiaAhmedBushra/b12-a09-green-nest.git
cd b12-a08-hero-apps

# Install client dependencies
npm install
````
  
## After Installation

Follow these steps to run the project locally:

Start the frontend client
```bash
npm run dev
```
Then open your browser and visit:
http://localhost:3000
